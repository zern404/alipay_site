# Servicre - Многостраничный SEO-лендинг

Многостраничный SEO-лендинг для карго-доставки из Китая и финансовых услуг (пополнение Alipay, WeChat Pay, оплата инвойсов).

## Технологии

- **Next.js 14** (App Router) - для SEO и производительности
- **TypeScript** - для типобезопасности
- **Tailwind CSS** - для стилизации
- **React 18** - для UI

## Структура сайта

- `/` - Главная страница
- `/kargo-dostavka-iz-kitaya/` - Карго-доставка из Китая
- `/popolnenie-alipay/` - Пополнение Alipay
- `/popolnenie-wechat-pay/` - Пополнение WeChat Pay
- `/oplata-invoisov-v-kitae/` - Оплата инвойсов в Китае
- `/mezhdunarodnye-platezhi/` - Международные платежи
- `/otzyvy/` - Отзывы клиентов
- `/kontakty/` - Контакты

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000)

## Сборка для продакшена

```bash
npm run build
npm start
```

## SEO-оптимизация

- ✅ Мета-теги (title, description, keywords) для каждой страницы
- ✅ Open Graph теги
- ✅ Структурированные данные (JSON-LD)
- ✅ Семантическая разметка (h1, h2, h3)
- ✅ Внутренняя перелинковка между страницами
- ✅ ЧПУ URL на русском транслите
- ✅ Адаптивный дизайн
- ✅ Быстрая загрузка (Next.js оптимизация)

## Особенности

- **Многостраничная структура** - каждая страница = отдельный SEO-хаб
- **Полные тексты** - минимум 5-7 тыс. знаков на ключевые страницы
- **Живой контент** - деловой тон
- **CTA на Telegram** - везде призыв к действию
- **FAQ-блоки** - для лучшего SEO в Яндексе
- **Внутренняя перелинковка** - связи между страницами
